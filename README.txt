In order to run the application, you must install NodeJS first, preferably v20
Afterwards go to where server.js is located 
Type "cd book-store" (or another path your project located in)
Type "node server.js" or >npm run start
Head over to your browser and follow the link "localhost:3000" or "127.0.0.1:3000"
